# 🛍 E-commerce Customer Segmentation (RFM Analysis)

## 📌 Overview
This project segments customers based on purchasing behavior using RFM analysis.

## 🎯 Objectives
- Identify high-value customers
- Improve customer retention
- Enable targeted marketing strategies

## 🛠 Tools Used
- Python (Pandas)
- Excel
- Power BI

## 📂 Dataset
- 500,000+ transaction records
- Fields: CustomerID, InvoiceDate, Quantity, UnitPrice

## 🔄 Workflow
1. Data Cleaning (remove missing IDs, negative values)
2. Feature Engineering (TotalAmount)
3. RFM Calculation
4. Customer Segmentation

## 📊 Key Insights
- Majority of customers are lost
- Loyal customers contribute high revenue
- Potential loyalists can be targeted for growth

## 📈 Dashboard Features
- Customer Segment Distribution
- Revenue by Segment
- Customer Behavior Analysis

## 📌 Conclusion
RFM segmentation helps businesses improve retention and revenue.
